# Personal Portfolio

A minimalist and fast portfolio for **Vedant Parmar** showcasing projects in AI, computer vision, and Flutter/mobile development.

## 🚀 Features
- Responsive layout
- Accessible semantic structure
- AOS scroll animations (lightweight)
- Clean, maintainable CSS (single file)

## 📂 Project Structure
```
index.html      # Main HTML
style.css       # Styles
assets/         # Images & icons
```

## 🔧 Local Development
Just open `index.html` in a browser. No build step needed.

## 🛡️ Security / Best Practices
- All external links use `rel="noopener noreferrer"`.
- No third-party tracking scripts.
- Only two external CSS/JS CDNs (Bootstrap + AOS).
