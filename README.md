# Personal Portfolio

A minimalist and fast portfolio for **Vedant Parmar** showcasing projects in AI, computer vision, and Flutter/mobile development.

## 🚀 Features
- Responsive layout
- Accessible semantic structure
- AOS scroll animations (lightweight)
- Lazy-loaded project media
- Open Graph + Twitter meta tags (add your live URL)
- Clean, maintainable CSS (single file)

## 📂 Project Structure
```
index.html      # Main HTML
style.css       # Styles
assets/         # Images & icons
```

## 🔧 Local Development
Just open `index.html` in a browser. No build step needed.

## 🌐 Deployment (GitHub Pages)
1. Create a new GitHub repository (e.g. `portfolio` or `vedant-portfolio`).
2. Push the code (see commands below).
3. In repo settings -> Pages -> Select `main` branch `/ (root)`.
4. Save; wait for deployment (1–2 min). Your site will be at:
   `https://<your-username>.github.io/<repo-name>/`
5. Update the `og:url` meta in `index.html` with the final URL.

## 🛡️ Security / Best Practices
- All external links use `rel="noopener noreferrer"`.
- No third-party tracking scripts.
- Only two external CSS/JS CDNs (Bootstrap + AOS).

## 📝 Update Open Graph Meta
In `index.html`, replace:
```
<meta property="og:url" content="https://YOUR_DOMAIN_OR_GITHUB_PAGES_URL" />
```
with your actual deployed URL.
